Fontes em CSS

  • CSS

    • Uma vez que nem todas as fontes existentes estão instaladas nos computadores dos usuários (existem milhares de fontes e a maioria delas não é gratuita), CSS fornece um mecanismo para compensar esta lacuna. Você define uma lista de fontes para seu documento a serem usadas na ordem que você definiu (se a primeira da lista não estiver disponível no computador do usuário usar a segunda e assim por diante) e a última fonte da lista deverá ser uma fonte genérica

Cores em CSS RGB

  • CSS RGB

    • A função rgb, define cores usando o modelo Vermelho-verde-azul (RGB). Um valor de cor RGB é especificado com: rgb (vermelho, verde, azul). Cada parâmetro define a intensidade dessa cor e pode ser um inteiro entre 0 e 255 ou um valor percentual (de 0% a 100%).

Cores em CSS RGBA

  • CSS RGBA

    • A função rgba, define cores usando o modelo Vermelho-verde-azul-alfa (RGBA). Os valores de cor RGBA são uma extensão dos valores de cor RGB com um canal alfa - que especifica a opacidade da cor.

Posicionamento

  • Position Absolute

    • O elemento é posicionado em relação ao primeiro elemento ancestral posicionado (não estático)

  • Position Relative

    • O elemento está posicionado em relação à sua posição normal, então "left: 20px" adiciona 20 pixels à posição LEFT do elemento

  • Float

    • A propriedade float especifica como um elemento deve flutuar.

  • Position Fixed

    • O elemento é posicionado em relação à janela do navegador

  • Position sticky

    • Um elemento adesivo alterna entre relativo e fixo, dependendo da posição de rolagem. Ela é posicionada relativa até que uma determinada posição de deslocamento seja atendida na viewport - então ela "gruda" no lugar (como posição: fixa).
